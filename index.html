Personal Projects

Project Golden Goose

The goal for Project Golden Goose is to serve as copy of the website DNDbeyond, but a bit more accessible for the average player! Although the DNDbeyond website works well enough for the current Fifth Edition ruleset (or 5e for short), that is not to say that it is flawless. Our main gripes with the site lie with paywalled content that is readily avaliable online, an inventory system that can be improved upon, character sheet limits with free accounts, and most importantly to us, a solid lack of homebrew implementation! Through this project we hope to supply our close friends with a better platform to run our campaigns with the inclusion of a more optimized homebrew experience! We are hoping to have a fully deployable website in the near future. Although we know we cannot fully release this due to the potential wrath of Wizards of the Coast, being able to use our web development skills to create a better experience for our close friends is all we can hope for. We have thus far used HTML, CSS and JavaScript to create the current foundation of our site, although it is still in very early development. Frupal

Frupal is a small game developed entirely in Python using its built in GUI Tkinter! The object of the game is fairly simple: navigate throughout the map and attempt to claim all the diamonds without running out of energy! There are items to encounter that will help you on your journey, obstacles that will make it harder for you to traverse through the map, and different types of terrain that will consume more energy for each cell you step in! Developed over the summer in my CS314 class, while the game itself was fairly easy to implement as a team, it mainly acted as an exercise in software development within a team of people. Following the SCRUM development process, the goal of the class was to demonstrate how our team would be working in industry by following this design process.
